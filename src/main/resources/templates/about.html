Опыт работы с технологиями

В ходе разработки данного проекта были использованы следующие технологии и инструменты:

  • Java 21 - основной язык разработки.
  • Spring Boot 3 - фреймворк для создания веб-приложения.
  • Spring Security - обеспечение безопасности и разграничения прав доступа.
  • Spring Data JPA & PostgreSQL - работа с базой данных.
  • Thymeleaf - шаблонизатор для генерации HTML-страниц.
  • Bootstrap 5 - адаптивная верстка и стилизация интерфейса.
